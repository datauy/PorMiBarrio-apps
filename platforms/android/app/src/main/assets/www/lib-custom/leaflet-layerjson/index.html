Leaflet Dynamic JSON Layer

Simple way for transform any JSON data source in a Leaflet Layer!

Other useful stuff for Web Mapping...

Features

  • Load JSON data via AJAX,JSONP or custom function
  • Request data by bounding box
  • Caching System for reduce Ajax/Jsonp requests
  • Pre-filter JSON data from third-party jsonp services
  • Events: dataloading, dataloaded
  • Customize popup and icon marker
  • Customize marker builder
  • Using exists layer

Options

  • min shift before request data
  • new requests only if comes out of bounds
  • precision digit for coordinates
  • and much more...

Examples

  • Simple Ajax
  • JSONP OSM Nominatim
  • Data by Callback
  • Overpass API
  • Remote Fixed Data

Usage

L.layerJSON({url: "search.php?lat1={lat1}&lat2={lat2}&lon1={lon1}&lon2={lon2}" }).addTo(map);
